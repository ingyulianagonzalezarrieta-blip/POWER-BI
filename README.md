# 📊 Portafolio Power BI

> *"Los datos cuentan historias. Aquí aprendo a contarlas."*

Bienvenido a mi portafolio de análisis de datos en Power BI.
Este espacio reúne proyectos donde transformo datos reales en
dashboards interactivos y visualmente claros, aplicando un proceso
completo que va desde la adquisición y limpieza de datos hasta
la construcción de visualizaciones que facilitan la toma de decisiones.

Cada proyecto utiliza fuentes de datos oficiales y reales,
priorizando la calidad, la trazabilidad y la interpretación
significativa de la información.



## 🎯 ¿Qué encontrarás aquí?

- 📥 **Datos reales** provenientes de fuentes oficiales y verificables
- 🧹 **Proceso de limpieza** documentado paso a paso
- 📊 **Dashboards interactivos** desarrollados en Power BI Desktop
- 📝 **Documentación clara** de cada proyecto y sus hallazgos
