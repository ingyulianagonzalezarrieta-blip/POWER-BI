# 📊 Dashboard Comercial 2023
## Comercializadora Central S.A.S — Sucre, Colombia

Dashboard interactivo desarrollado en Power BI con datos ficticios
basados en la operación real de una distribuidora de bebidas en el
departamento de Sucre, Colombia.

---

## 🎯 Objetivo

Visualizar y analizar el comportamiento comercial de una distribuidora
de bebidas (cerveza, agua, malta y Electrolit) con rutas de distribución
en los municipios de Caimito, La Unión y San Marcos, incluyendo sus
veredas y zonas rurales.

---

## 📈 Páginas del dashboard

### 1. Análisis de Ventas
- Ventas totales y utilidad del año
- Ventas mensuales 2023
- Ventas por producto
- Ventas por ruta de distribución
- Ventas por tipo de cliente

### 2. Análisis de Inventario
- Stock actual por producto
- Estado del inventario (Crítico/Normal/Exceso)
- Margen de ganancia por producto
- Rotación de inventario en días

### 3. Indicadores Comerciales
- Cumplimiento promedio de metas
- Clientes atendidos
- Ticket promedio
- Cumplimiento por vendedor
- Pedidos por ruta
- Meta vs ventas reales por mes

## 🍺 Productos analizados

- **Cerveza:** Águila, Póker, Club Colombia, Costeña
- **Agua:** Cristal 600ml, Cristal 1.5L
- **Malta:** Pony Malta, Malta Leona
- **Electrolit:** Electrolit x12

## 🛠️ Proceso

1. Generación de datos ficticios con Python
2. Importación y limpieza en Power Query
3. Modelado y relaciones entre tablas
4. Construcción del dashboard en Power BI Desktop
